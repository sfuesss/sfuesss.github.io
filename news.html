Linux Workshop

The Linux Workshop is a series of workshops that guide you through everything you need to know about linux from using the command line to superuser skills. The workshop is based on Ubuntu Linux 22.04. For Windows users, download VirtualBox. if you have an intel based mac, we suggest Virtualbox; however, if you have an ARM based mac (M1/M2 Chip), we suggest UTM. The Ubuntu ISO will redirect you to the official Ubuntu download page. We suggest downloading 22.04 as that is the LTS version they currently have.
